CoAtNet：Marrying Convolution and Attention for All Data Sizes[Google 202109]

发表于 2025-11-06 更新于 2025-11-08 分类于 笔记 阅读次数：
本文字数： 2.1k 阅读时长 ≈ 2 分钟

摘要

当时的Transformer在计算机视觉领域仍然达不到卷积的state-of-the-art效果。Transformer有更大的model capacity，但由于没有inductive bias缺少泛化。CoAtNets是一种混合模型。1.由深度可分离卷积以及自注意力，通过相对注意力结合。2.垂直结构堆叠卷积以及注意力，在提升模型的泛化能力（generalization）、表达能力（capacity）以及计算效率（efficiency）方面表现出惊人的效果。使用了ImageNet-21K以及JFT-300M数据集。

该方法通过混合，达到了ViT-huge的水平，且数据少了23倍。

Introduction

Vit模型在大数据集的训练效果很好，但是在low data regime的效果不如卷积网络。本文主要考虑两个关键部分，generalization以及model capacity。本文系统地讨论了混合问题。一般来说卷积泛化能力更好，收敛更快，transformer有更好的模型能力。主要的问题是如何结合，追求trade-offs between accuracy and efficiency。文中发现了两个关键点：1.深度可分离卷积可以有效的融入attn，通过相对注意力。2.堆叠可以很有效的提高泛化以及模型能力。

Model

融合卷积与自注意力

有两个关键的相似点：

  • 卷积和Transformer里的FFN都用了‘反瓶颈’模块，通道变成4倍之后还原。
  • 都用了pre-defined感受野下的加权求和（卷积以及attn）

一些希望得到的好的特性：

  • 深度可分离卷积的权重与输入无关，而注意力的权重是全局相关的，可以得到不同位置的空间关系，这是我们处理的时候希望得到的high-level concepts。不过这一点在数据量小的时候，也容易引入过拟合。
  • 卷积的权重$w_{i-j}$只依赖于$i-j$而与实际的位置无关，也是由此得到了平移不变性。由于使用了绝对位置编码，标准的ViT缺少这种性质，这也解释了为什么本文的方法在数据量不够大的时候，比ViT效果更好。
  • 感受野的大小是很重要的一个因素，更大的感受野提供了更多的contextual information，提高model capacity。这也是为什么希望将注意力用于视觉领域。这一点也带来了更多的计算量，是空间大小的二次方关系。

一种直接的结合方法就是，在softmax之前或者之后，直接加上卷积核的权重$w_{i-j}$.

文中还提到，在新的公式里，$w_{i-j}$被简化为一个scalar而不是一个向量，应该是说，不同的channel用的一样的$w_{i-j}$所以使得参数量不会爆炸（Scalar formulation）。文中后续的方法是用的（3）中的$y^{pre}$进行计算，pre-normalization。

垂直堆叠设计

考虑到图像通常比较大，实际上（3）的计算会比较复杂，有以下3个方案：

(A) 进行下采样（down-sampling），再施加全局相对注意力；

(B) 采用局部注意力（local attention），即将注意力的全局感受野 G 限制为一个局部区域 L，类似于卷积操作；

(C) 将具有二次复杂度的 Softmax 注意力替换为某种线性注意力（linear attention）变体，使其计算复杂度仅与空间尺寸线性相关。

文中试了一下，bc方案都不太好，后续都用A方案进行处理。下采样可以两个方案，一个是直接Vit一样，分块比较激进的方法进行卷积，另外一个就是多阶段的池化例如ConvNet的方法。为此本文提出了五种用于对比的模型结构，有四种结构都是conv逐渐降低维度的，因为采用不同的卷积以及自注意力顺序所以一共有四种不同的对比结构，不过卷积都在transformer之前出现。

  • 全transformer
  • C-C-C-C
  • C-C-C-T
  • C-C-T-T
  • C-T-T-T

五个模型都用了近似的参数量，没有用数据增广以及正则化。ImageNet-1K (1.3M) and JFT (>300M) dataset for 300 and 3 epochs。验证泛化能力（验证集准确率）以及model capacity。

  • 以ImageNet结果为例，首先是泛化能力，可以看到卷积越多，泛化能力越好，也就是在验证集上的准确率越高。

后续实验为了选出最好的模型，比较C-C-T-T and C-T-T-T，额外加了一个transferability实验，先在JFT数据集预训练，然后再ImageNet-1K上微调30epochs。发现C-C-T-T效果更好一点。也将这个结构作为CoAtNet的结构。

注意文中的相对注意力不是交叉注意力，相对注意力依然是自注意力的一种，只是在计算注意力得分的时候，会用到一些相对位置的信息（不同的位置编码形式）。
